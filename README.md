# АвтоСервис — Рекомендации по ТО + DeepSeek AI

Настольное приложение для расчёта рекомендаций по техническому обслуживанию автомобиля с интеграцией DeepSeek AI.

![Python](https://img.shields.io/badge/Python-3.10+-blue?logo=python&logoColor=white)
![Platform](https://img.shields.io/badge/Platform-Windows%20|%20macOS%20|%20Linux-lightgrey)
![License](https://img.shields.io/badge/License-MIT-green)

## Возможности

- **Расчёт рекомендаций** по 13 ключевым узлам автомобиля (ГРМ, масло, тормоза, АКБ, шины и др.)
- **Три уровня приоритета** с цветовой индикацией: 🔴 высокий, 🟠 средний, 🔵 рутинный
- **Учёт марки авто** — корректировка интервалов для японских, немецких, корейских, российских, французских марок
- **Учёт типа двигателя** — бензин/дизель (свечи зажигания vs накаливания, интервал масла)
- **DeepSeek AI** — глубокий персональный анализ нейросетью с типичными проблемами модели и стоимостью работ
- **Потоковый вывод AI** — ответ появляется в реальном времени

## Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/AI_Car_Detail.git
cd AI_Car_Detail

# Запустить (Python 3.10+, без внешних зависимостей)
python car_maintenance.py
```

### Настройка DeepSeek AI (опционально)

1. Получите API ключ на [platform.deepseek.com](https://platform.deepseek.com)
2. В приложении вставьте ключ в поле **«API ключ»** → **«Сохранить ключ»**
3. Нажмите **«Получить анализ от ИИ»** после расчёта рекомендаций

Ключ сохраняется локально в `config.json` (файл добавлен в `.gitignore`).

| Масло АКПП / вариатора | — | — |
